### 準備
---
.envファイルプロジェクトのルートディレクトリに作成し配置。以下を記述。
```
DB_HOST=接続ホスト
DB_NAME=データベース名
DB_PORT=ポート番号
DB_USER=ユーザー名
DB_PASS=パスワード
TZ=タイムゾーン
```
例
```
DB_HOST=localhost
DB_NAME=hoge
DB_PORT=3306
DB_USER=fuga
DB_PASS=piyo
TZ=Asia/Tokyo
```
### docker
---
コンテナ起動
  
```docker-compose up -d```
  
コンテナ停止
  
```docker-compose stop```
  
コンテナ削除
  
```docker-compose down```
  
コンテナ更新
  
```docker-compose up -d --build```
  
MySQLにアクセス
  
```docker-compose exec db bash```
  
MySQLにログイン
  
```mysql -u [username] -p```

### DBmigrate
---
必要パッケージ
```
pip install simple-db-migrate
pip install python-dotenv
pip install mysqlclient
```
./migrateにて以下のコマンドを実行
  
```db-migrate```
